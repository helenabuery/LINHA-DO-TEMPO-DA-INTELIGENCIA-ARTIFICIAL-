# LINHA-DO-TEMPO-DA-INTELIGENCIA-ARTIFICIAL-
## Linha do tempo da Inteligência Artificial 

Nessa atividade foi desenvolvida para a disciplina de Desenvolvimento Web, com o objetivo de criar uma linha do tempo interativa sobre a história da Inteligência Artificial.

Utilizamos ferramentas de Inteligência Artificial para auxiliar na construção do conteúdo e posteriormente implementar a aplicação utilizando HTML, CSS e JavaScript.

## Metodologia: 
O desenvolvimento do projeto ocorreu em três etapas principais:

###  – Geração do Modelo Inicial
Foi utilizado uma ferramenta de Inteligência Artificial para gerar um modelo inicial de mapa conceitual e estrutura da linha do tempo com base em referências históricas sobre IA.

###  – Refinamento e Estruturação
Após testes com diferentes ferramentas de IA, o conteúdo foi revisado, reorganizado e adaptado. Também foram feitas melhorias na estrutura HTML, estilização com CSS e ajustes na lógica em JavaScript.

###  – Implementação de Funcionalidades
Foram adicionados recursos de acessibilidade e interatividade:
- Controle de tamanho da fonte
- Modo claro e modo escuro
- Barra de pesquisa para marcos históricos

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
- Ferramentas de Inteligência Artificial para apoio na construção do conteúdo
